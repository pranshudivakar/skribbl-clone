# 🎨 Skribbl.io Clone

A real-time multiplayer drawing and guessing game built with React, Node.js, and Socket.IO.

## 🚀 Live Demo
[Live URL](https://your-skribbl-clone.onrender.com)

## 🎯 Features

### Core Features
- ✅ Create/Join rooms with unique codes
- ✅ Real-time drawing synchronization
- ✅ Turn-based gameplay with rotating drawers
- ✅ Word guessing with scoring system
- ✅ Lobby system with ready-up
- ✅ Configurable room settings
- ✅ Private rooms

### Drawing Tools
- Multiple colors
- Brush size adjustment
- Clear canvas
- Undo last stroke

### Game Features
- Timed rounds
- Hints system
- Score tracking
- Leaderboard
- Chat system

## 🛠️ Tech Stack

### Frontend
- React 18
- Socket.IO Client
- HTML5 Canvas
- Tailwind CSS
- Vite

### Backend
- Node.js
- Express
- Socket.IO
- Custom WebSocket handlers
